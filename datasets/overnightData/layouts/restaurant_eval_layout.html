Instructions

Write a few commands that a robot (placed in a sandbox with blocks) can execute. Below are a few properties of this sandbox:

  • There is an assortment of blocks/bricks/pyramids/tetrahedra/prisms
  • Each of these blocks has a color (red/green/blue), a length, a height, and a width all between 0 and 5 (units not specified)
  • You can query the robot for how many blocks, create a block, or remove a block
  • You can filter your queries in various ways (see examples)
  • Your commands must be original and can NOT simply be a copy of an example command

Examples:

  • what is the number of red blocks?
  • how many prisms have height greater than 3?
  • what is the longest green pyramid left of the red block
  • create a red brick
